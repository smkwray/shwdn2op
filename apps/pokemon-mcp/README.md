# Local Pokémon MCP server

This is a small starter MCP server for Pokémon knowledge.

It is intentionally lightweight:

- enough to help the model reason about types, a few species, a few moves, and some sample common sets
- easy for Codex to extend
- not yet a complete Pokédex or damage engine

## Current tools

- `lookup_pokemon`
- `lookup_move`
- `type_effectiveness`
- `common_sets`

## Long-term direction

This server should eventually be replaced or extended with fuller deterministic data and calculations.
